Visualizing Trends in Supercomputing

Blake Haugen and Nick McNutt

Performance Over Time

The performance plot shows that the speed of supercomputers has been increasing exponentially for the last 20 years. The plot shows the performance of the #1 system on the Top 500 list at each point in time as well as the performance of the #500 machine and the sum of the performance of all machines. The performance is measured using the Linpack benchmark. This plot leaves many people wondering how long the exponential growth in performance will continue.

Source: Top 500

Number of Cores Over Time

As the performance of supercomputers has grown over time, so have the number of cores present in each system. This growth in the number of cores can make parallel programming a real challenge. It is interesting to see the rather flat section of the graph from 1997 to 2004. Around 2004 the number of cores begins to dramatically increase with each releas of the list. This corresponds directly with the point in time where manufactures stabilized the clock frequency of the processor but began manufacturing multicore chips.

Source: Top 500
